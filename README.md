# **Hello World Script**

## Description
A simple script that types in the console 'Hello World!'

## Install Instructions
To install, make sure to install every file in the repository (This README, the package.json, and the index.js). 
If you want to run this in your coding environment (e.g. Visual Studio Code), ensure you have Node.js installed.

## Run Instructions
To run this program, type in your terminal of choice node index.js to run the script.
